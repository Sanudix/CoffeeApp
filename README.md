# Мобильное приложение для кофейни

## Описание
Приложение написано с использованием Jetpack Compose для построения UI. За основу взят [дизайн из Figma](https://www.figma.com/design/Lqh7CJasZFxRFWIx2gD2ZL/Coffee-Shop-App-UI-UX-Design--Community-?node-id=1-323&t=4lvVSngi5I84bJpR-1).
Проект создан для демонстрации навыков работы с Jetpack Compose, а также работы с Figma.
